### Data Analysis for Chart One
- Available data comes from the Data Source 2.
- Created with Datawrapper.
- This chart was created to sort of counter and complete Data Source 3, a report shared with reporters to hail the dept's successful policy. During my analysis, I discovered an incomplete set of data manipulates the key data points and they were not disclosed at all in the existing articles. I chose to omit those false data points and let it be known that the original doata is partial and some numbers are probable estimates.
- While recalculating the numbers myself for sanity check, I found the Data Source 3 had slightly spun the numbers to make the numbers look good, such as the start point is a tad bigger while the end point is a bit smaller for more dramatic change. For the chart, more precise numbers were drawn to paint a clearer picture.

### Data Analysis for Chart Two  
- Available data comes from Data Source 2 and 3.
- Created with Datawrapper.
- Number and prices of green/gel packs, levy per Kilogram, average weight per pack, price index followed Data Source 2 and 3.
- Environmental levy per pack formula: 313 (levy per Kilogram in KRW) * 0.5 (average weight per pack in KG) * 1.1441 (price index)
- Total costs = (N of non-gel packs * price per non-gel pack) + (N of gel packs * price per gel pack) + (Environmental levy per pack * N of gel packs)
- Conversion to USD from KRW is done on March 2, 2026, with Google Sheet's googlefinance function.
